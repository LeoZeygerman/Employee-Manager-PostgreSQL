Employee Manager

CLI-приложение для управления сотрудниками, написанное на Python с использованием PostgreSQL.

Возможности

* ➕ Добавление нового сотрудника
* ❌ Удаление сотрудника
* 👥 Просмотр списка всех сотрудников
* 🔍 Поиск сотрудника:
    * по ID
    * по имени
* 💰 Добавление бонусов сотруднику
* ⚠️ Добавление штрафов сотруднику
* 📝 Добавление причины штрафа или бонуса
* 📜 Просмотр полной истории штрафов и бонусов сотрудника
* 💵 Расчет итоговой выплаты с учетом всех бонусов и штрафов

Используемые технологии

* Python 3
* PostgreSQL
* psycopg2

База данных

Проект использует PostgreSQL.

Основные таблицы:

* employee — информация о сотрудниках
* salary — история бонусов и штрафов

Связь осуществляется по worker_id.

Как запустить

1. Установить PostgreSQL.
2. Создать базу данных.
3. Установить зависимости:

pip install psycopg2

4. Указать свои параметры подключения в database.py:

host='localhost'
database='employee-manager'
user='postgres'
password='******'
port='5432'

5. Запустить приложение:

python main.py
